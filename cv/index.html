Projects

  • Learn to Merge Tokens for Communication-Efficient Collaborative Occupancy Prediction

    Ongoing research on communication-efficient collaborative 3D semantic occupancy prediction for autonomous driving, focusing on tokenized scene representation, spatio-temporal memory, and task-aware communication under bandwidth constraints.

    • Designed a collaborative occupancy prediction framework with tokenized spatio-temporal memory for multi-agent scene understanding
    • Developed communication-aware token merging strategies to reduce transmission redundancy while preserving task-critical information
    • Implemented request-conditioned communication, multi-agent feature alignment, and Transformer-based temporal modeling in a unified 3D world coordinate system
    • Built large-scale training and evaluation pipelines on collaborative perception datasets, analyzing communication cost, memory efficiency, and occupancy prediction accuracy
  • Rate-Distortion in Efficient Multi-Agent Perception: A Unified Framework for Communication and Memory Optimization

    Ongoing research on a unified rate-distortion perspective for efficient multi-agent perception, jointly optimizing communication bandwidth and spatio-temporal memory usage in collaborative autonomous driving systems.

    • Formulated multi-agent perception as a joint communication-memory optimization problem under rate-distortion constraints
    • Investigated unified token selection, token compression, and memory update strategies for balancing perceptual utility against resource cost
    • Developed theoretical and algorithmic designs for communication-efficient and memory-aware perception pipelines in collaborative driving scenarios
    • Planned empirical evaluation on collaborative occupancy prediction benchmarks with analysis of accuracy–bandwidth–memory trade-offs
  • Federated Learning for Medical Image Segmentation with Domain Generalization

    Research project on privacy-preserving medical image segmentation under cross-institution domain shift, combining federated learning, domain generalization, and cross-domain feature adaptation.

    • Developed a federated learning framework for multi-center medical image segmentation without sharing raw patient data
    • Introduced domain style transfer and feature alignment strategies to improve model robustness across heterogeneous medical datasets
    • Implemented collaborative optimization across distributed clients and evaluated cross-domain generalization performance
    • Studied the interaction between privacy-preserving training and domain shift mitigation for robust medical image analysis
  • Jetson-Based Autonomous Driving System

    Embedded autonomous driving project built on NVIDIA Jetson, integrating visual perception, traffic light recognition, obstacle detection, path planning, and behavior control in real time.

    • Trained and deployed lightweight object detection models (YOLO-based) for traffic light recognition and obstacle perception on Jetson hardware
    • Implemented lane following, rule-based path planning, and crosswalk-aware speed control for autonomous navigation
    • Optimized perception and control modules for real-time inference on embedded GPU platforms
    • Integrated perception, planning, and control into a complete autonomous driving pipeline for structured road environments
  • NAO Robot Soccer System

    Embodied AI project based on the NAO humanoid robot platform, integrating computer vision, motion control, and behavior planning for autonomous robotic soccer.

    • Developed real-time visual perception modules for soccer ball detection, localization, and tracking using onboard camera inputs
    • Implemented robot motion control for walking, turning, target alignment, and kicking using NAO control APIs
    • Designed behavior logic for autonomous soccer play, including target search, approach, positioning, and kick execution
    • Built an embodied perception-action pipeline combining sensing, decision-making, and control in a robotic system
  • Embedded Face Recognition Access Control System

    Edge AI access control system based on STM32 microcontrollers, combining lightweight face recognition, IoT communication, and Android-based remote control.

    • Designed and deployed a lightweight face recognition pipeline on STM32 embedded hardware using a compressed CNN-based model for resource-constrained inference
    • Applied model lightweighting techniques such as pruning, quantization, and embedded deployment optimization for real-time on-device recognition
    • Integrated ESP8266 and Baidu Cloud IoT platform for remote communication, device monitoring, and access management
    • Developed an Android application for remote door control, user interaction, and cloud-connected system management
